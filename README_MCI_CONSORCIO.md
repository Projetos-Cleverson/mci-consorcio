# MCI Consórcio Imobiliário — Base visual do MCI Imobiliário

Esta versão usa a estrutura visual do projeto **MCI — Mapa da Compra Imobiliária** e adapta os textos, perguntas e perfis para o produto:

**MCI Consórcio Imobiliário — Diagnóstico de Compra Planejada**

## Principais ajustes

- Landing page reaproveitada do MCI Imobiliário, com textos focados em consórcio.
- Quiz com 18 perguntas de aderência ao consórcio.
- 6 perfis de resultado:
  - Consórcio Planejado
  - Lance Estratégico
  - Estratégia Patrimonial
  - Troca ou Upgrade Imobiliário
  - Preparação para Consórcio
  - Análise de Aderência
- Salvamento adaptado para a tabela `mci_consorcio_leads` no mesmo Supabase.
- Captura de parceiro por URL: `?partner=empresa-a`.

## Observação técnica

Para reaproveitar rapidamente a estrutura visual e administrativa do MCI Imobiliário, os códigos internos dos perfis foram mantidos e apenas mapeados para os nomes comerciais do MCI Consórcio.
